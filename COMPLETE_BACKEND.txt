   bash setup.sh

2. Configure .env:

💾 DATABASE SCHEMA
═════════════════════════════════════════════════════════════════════════════

hackathons table:
  • id (UUID, PK)
  • user_id (FK → auth.users)
  • name, url, location, mode, description
  • registration_deadline, submission_deadline (ISO timestamps)
  • status (interested|registered|completed|dropped)
  • created_at, updated_at

plans table:
  • id (UUID, PK)
  • hackathon_id (FK → hackathons)
  • user_id (FK → auth.users)
  • project_idea, tech_stack[], team_members[], notes, priority
  • created_at, updated_at

user_tokens table:
  • id (UUID, PK)
  • user_id (FK → auth.users)
  • token (unique)
  • created_at, expires_at
